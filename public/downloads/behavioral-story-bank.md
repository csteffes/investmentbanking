## Instructions

Build 5–8 stories before you rehearse any behavioral answers.
Each story should map to multiple question types.
Use STAR-L format: Situation → Task → Action → Result → Lesson.

## Anti-Pattern Checklist

Before you finalize each story, check for these common mistakes:

- [ ] Does the action section say "we" more than "I"? Fix it.
- [ ] Is the result vague ("it went well")? Add a number or specific outcome.
- [ ] Is the story longer than 90 seconds when spoken aloud? Trim it.
- [ ] Can you answer one follow-up question with equal specificity? Practice it.
